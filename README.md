# SimplePaint
## Descripción:
SimplePaint es un proyecto simple que implementa un entorno similar a Microsoft Paint, desarrollado utilizando la biblioteca Turtle de Python. Este es un clon básico de herramientas de dibujo, ideal para aprender conceptos de programación gráfica en Python.

## Características Principales

Interfaz gráfica simple con Turtle para dibujo libre.
Herramientas básicas de dibujo, como pincel.
Soporte para colores y grosores de línea.
Entorno interactivo para practicar programación gráfica.

(Nota: Detalles específicos de características pueden variar; consulta el código fuente para funcionalidades exactas.)
## Requisitos

- Python 3.x instalado.
- Biblioteca turtle (incluida en la biblioteca estándar de Python, no requiere instalación adicional).

## Instalación

### Clona el repositorio:
```bash 
git clone https://github.com/brayanvillca/simplePaint.git
```

### Navega al directorio:
```bash
cd simplePaint
```
### Ejecuta el script principal:
```python
python3 simplepaint.py
-
python simplepaint.py
```

## Uso

Ejecuta el script Python.
Se abrirá una ventana gráfica con Turtle.
Usa el mouse para dibujar en la pantalla.
Experimenta con comandos de Turtle para agregar funcionalidades como cambiar colores.
